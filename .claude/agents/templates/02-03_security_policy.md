# 데이터 수집 보안 정책

## 1. 수집 금지 항목 (PII)
| 항목 | 유형 | 이유 |
|------|------|------|
| 이름 | 직접 식별 | 개인정보보호법 |
| 전화번호 | 직접 식별 | 개인정보보호법 |
| 이메일 원문 | 직접 식별 | 개인정보보호법 |
| 카드번호 | 금융정보 | PCI DSS |

## 2. 익명화 필요 항목
| 항목 | 처리 방법 | 예시 |
|------|-----------|------|
| email | SHA-256 해싱 | user@example.com → a1b2c3... |
| IP 주소 | 마지막 옥텟 마스킹 | 192.168.1.xxx |

## 3. 데이터 보존 기간
| 데이터 유형 | 보존 기간 | 삭제 방법 |
|-------------|-----------|-----------|
| 원시 이벤트 | 24개월 | 자동 만료 |
| 집계 데이터 | 영구 | - |
| 사용자 속성 | 계정 삭제 시 | 요청 기반 |

## 4. 접근 레벨
| 레벨 | 접근 범위 | 해당 역할 |
|------|-----------|-----------|
| Level 1 | 원시 데이터 + 관리자 | 데이터팀 리드 |
| Level 2 | 원시 데이터 (읽기) | 데이터팀 |
| Level 3 | 집계 데이터 + 대시보드 | 개발팀, 마케팅팀 |
| Level 4 | 대시보드만 | 경영진 |
