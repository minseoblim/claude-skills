| 역할 | GA4 | BigQuery | 대시보드 | GTM |

## 최소 권한 원칙 적용
| 원칙 | 내용 |
|------|------|
| 기본 접근 | 필요 최소 권한만 부여 |
| 권한 승인 | PM 또는 데이터팀 리드 승인 |
| 임시 권한 | 기간 제한 + 자동 만료 |
| 분기 리뷰 | 불필요 권한 정리 |
