# RACI 매트릭스

## 역할 정의
| 구분 | 설명 |
|------|------|
| R (Responsible) | 실행 담당자 |
| A (Accountable) | 최종 승인자 |
| C (Consulted) | 협의 대상 |
| I (Informed) | 통보 대상 |

## RACI 매트릭스

| 활동 | 데이터팀 | 개발팀 | 마케팅팀 | PM | 비고 |
|------|----------|--------|----------|-----|------|
| 이벤트 정의 | R | C | C | A | |
| GTM 태깅 구현 | C | R | I | A | |
| 지표 정의 | R | I | C | A | |
| QA 검증 | R | C | I | A | |
| 대시보드 개발 | R | I | C | A | |
| UTM 규칙 관리 | C | I | R | A | |
| 데이터 스키마 변경 | R | C | I | A | |
| 프로덕션 배포 승인 | C | C | I | A | |

## 에스컬레이션 경로
1단계: 담당자(R) → 승인자(A)
2단계: 승인자(A) → PM/프로젝트 리드
