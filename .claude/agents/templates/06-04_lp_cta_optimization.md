# LP/CTA 최적화 가이드

## 1. LP 유형별 구조

### 유형 A: 리드 생성형 (상담/견적)
| 섹션 | 내용 | 비고 |
|------|------|------|
| 히어로 | 핵심 가치 제안 + 메인 CTA | Above the fold |
| 문제 제기 | 타겟의 Pain Point 공감 | |
| 솔루션 | 서비스가 해결하는 방법 | |
| 사회적 증거 | 후기, 수치, 로고 | |
| CTA 반복 | 폼 또는 버튼 | |
| FAQ | 주요 의문 해소 | |

### 유형 B: 전환형 (가입/구매)
| 섹션 | 내용 | 비고 |
|------|------|------|
| 히어로 | 핵심 가치 제안 + 메인 CTA | Above the fold |
| 혜택 | 주요 기능/혜택 나열 | |
| 사회적 증거 | 후기, 수치 | |
| 가격/플랜 | 명확한 옵션 제시 | |
| CTA 반복 | 버튼 | |
| 보장 | 환불 정책 등 리스크 해소 | |
