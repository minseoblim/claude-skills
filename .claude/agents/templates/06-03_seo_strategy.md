# SEO 전략

## 1. 기술적 SEO 체크리스트
| 항목 | 상태 | 우선순위 | 설명 |
|------|------|----------|------|
| sitemap.xml | ☐ | 상 | 검색엔진 크롤링 기반 |
| robots.txt | ☐ | 상 | 크롤링 허용/차단 설정 |
| Core Web Vitals | ☐ | 상 | LCP, FID, CLS 최적화 |
| HTTPS | ☐ | 상 | 보안 연결 |
| 모바일 최적화 | ☐ | 상 | 반응형/모바일 퍼스트 |
| 구조화 데이터 (Schema) | ☐ | 중 | 리치 스니펫 |
| 메타 태그 최적화 | ☐ | 상 | title, description |
| 이미지 alt 태그 | ☐ | 중 | 이미지 검색 최적화 |
| 내부 링크 구조 | ☐ | 중 | 페이지 간 연결성 |
| 페이지 로딩 속도 | ☐ | 상 | 3초 이내 목표 |
