# 유저 플로우 다이어그램

## 1. 전체 플로우

[유입 경로]
├── 검색광고 (Google/Naver SA)
├── 디스플레이/SNS 광고 (Meta/GDN)
├── 오가닉 검색 (SEO)
├── 직접 유입 (Direct)
└── 추천/리퍼럴

    ↓ Landing

[랜딩 단계]
├── 메인 페이지
├── 캠페인 전용 LP
└── 제품 상세 페이지

    ↓ Explore

[탐색 단계]
├── 카테고리 탐색
├── 상품/서비스 비교
├── 후기/리뷰 열람
└── 가격 확인

    ↓ Convert

[전환 단계]
├── 회원가입
├── 상담 신청 / 견적 요청
├── 장바구니 추가
└── 구매/결제 완료

    ↓ Retain

[재방문/유지]
├── 재방문
├── 재구매
├── 리뷰 작성
└── 추천/공유

## 2. 단계별 이벤트 후보

| 단계 | 액션 | 이벤트 후보 | 유형 |
|------|------|-------------|------|
| 유입 | 페이지 첫 방문 | page_view | 자동 |
| 탐색 | 상품 열람 | product_view | 수동 |
| 전환 | 회원가입 완료 | signup_completed | 수동 |
