## 1. 식별자 체계

| 식별자 | 설명 | 생성 시점 | 지속 범위 | 저장 위치 |
|--------|------|-----------|-----------|-----------|
| user_id | 회원 고유 ID | 회원가입 시 | 영구 | 서버 DB |
| device_id | 기기 고유 ID | 첫 방문 시 | 기기 단위 | 쿠키/로컬스토리지 |
| session_id | 세션 ID | 방문 시작 시 | 세션 단위 | 세션 스토리지 |
| ga_client_id | GA4 클라이언트 ID | GA4 초기화 시 | 브라우저 단위 | GA4 쿠키 |
