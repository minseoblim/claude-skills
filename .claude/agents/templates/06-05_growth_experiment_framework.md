# 그로스 실험 프레임워크

## 1. 실험 프로세스
발굴 → 가설 수립 → 우선순위 → 설계 → 실행 → 분석 → 확장/폐기 → 반복

## 2. 가설 템플릿

**형식**: "만약 {변수}를 {변경 내용}하면, {측정 지표}가 {예상 변화}할 것이다. 왜냐하면 {근거}이기 때문이다."

| 항목 | 내용 |
|------|------|
| 가설 ID | EXP-001 |
| 문제 | (데이터에서 발견한 문제) |
| 가설 | (위 형식) |
| 변수 | (변경할 요소) |
| 측정 지표 | (Primary KPI) |
| 성공 기준 | (예: 전환율 +10%) |
| 최소 샘플 | (통계적 유의성 확보 기준) |
