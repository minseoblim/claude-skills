# CRM 플레이북

## 1. 고객 생애주기 시나리오

| 시점 | 채널 | 메시지 | 목적 | KPI |
|------|------|--------|------|-----|
| D+0 | 이메일 | 환영 메일 + 서비스 안내 | 첫 액션 유도 | 오픈율, 클릭율 |
| D+1 | 알림톡 | 주요 기능 안내 | 활성화 | 클릭율 |

## 2. 채널별 운영 가이드
| 채널 | 강점 | 적합 용도 | 발송 빈도 제한 | 비고 |
|------|------|-----------|---------------|------|
| 이메일 | 상세 콘텐츠 | 뉴스레터, 가이드 | 주 2회 이하 | 수신 동의 필수 |
| 카카오 알림톡 | 높은 도달률 | 알림, 리마인더 | 주 1회 이하 | 템플릿 승인 필요 |
| 앱 푸시 | 즉시성 | 긴급 알림, 프로모션 | 일 1회 이하 | 수신 허용 필요 |
| 문자 (LMS) | 보편적 도달 | 중요 공지 | 월 2회 이하 | 비용 고려 |

## 3. 메시지 발송 피로도 관리
| 규칙 | 내용 |
|------|------|
| 일일 최대 | 채널 합산 2건 이하 |
| 주간 최대 | 채널 합산 5건 이하 |
| Quiet Hours | 21:00 ~ 08:00 발송 금지 |
| Opt-out | 수신 거부 즉시 반영 |
| 우선순위 | 거래성 > 정보성 > 마케팅 |

## 4. NPS 측정 프레임
| 항목 | 내용 |
|------|------|
| 발송 시점 | 핵심 전환 후 D+7 |
| 측정 주기 | 분기별 |
| 질문 | "이 서비스를 주변에 추천할 의향이 얼마나 되시나요? (0-10)" |
| 분류 | Promoter (9-10), Passive (7-8), Detractor (0-6) |
| 산식 | NPS = %Promoter - %Detractor |
| 후속 액션 | Promoter → 리퍼럴 유도, Detractor → 이탈 방지 |
